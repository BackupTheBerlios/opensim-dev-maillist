At some point in a distant past we (libomv) nuked support for xmlrpc logins
in favor of LLSD logins, It was brought back into the fold a couple
versions ago using an unmodified version from
http://xmlrpccs.sourceforge.net/ and is currently the default method used
for logins (although LLSD logins are still supported). It is used also by
GridProxy/WinGridProxy with no changes for proxying login data.
