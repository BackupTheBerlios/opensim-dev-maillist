Hello folks,

Just thought I'd let you know about the progress of 0.6.8.RC1.  Currently, 
0.6.8.RC1 has the following serious Mantis issues.

0004439	
Cannot log in - [LLUDPSERVER]: Ignoring a repeated UseCircuitCode and 
[HTTPSERVER] forcible close errors

0004436	
Unable to notify Message Server about login. Presence might be borked for this user

0004351	
Messaging server gives Unable to connect to server http://userservername:8002. 
[^] Server not running?

0004420	
Viewer instantly terminates on logout - not properly shutting down

All 4 of these issues appear related to some intermittent problem communicating 
with the user service.  Current suspect is threadpool exhaustion on the user 
service.  Any investigative help here would be greatly appreciated.  This 
definitely happens under load.

The only other serious issue is

0004411	Running with HyperGrid enabled results in many errors

This appears to be caused by empty userInventoryURI and userAssetURI fields in 
the user profile.  Diva, you said that you knew of a simple solution for this? 
Any chance you could comment on mantis 4411?  Thanks.
