This is slightly off the topic of Facebook apps, so I started a new thread.

I just wanted to let you know that recently I have been testing with 
Wene Chke in France who has developed an "SLContact" component/module 
for Drupal and Joomla.  My tests have been with the Joomla 
version.  The module is now significantly more robust and has 
improved functionality to connect these content management data base 
systems (which run in MySQL for example) to Second Life, and 
(partially operational at the moment) to Opensim.  Along with some 
login/authorisation plugin ("SLAuth") and Drupal/Joomla/Wordpress 
blog posting LSL scripts in-world that work in Opensim and Second 
Life, there are a lot of opportunities opened up for collaboration in 
virtual worlds

http://joomlacode.org/gf/project/slcontact
http://joomlacode.org/gf/project/slauth
http://code.google.com/p/ssm2017binder/wiki/ssmBlogPosterEng

Others may be interested in these open source contributions from 
Wene, and perhaps able to contribute to the effort.  Ai
