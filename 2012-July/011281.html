  Just want to remind everyone: In January the parameter "region_id" was 
implemented as default to all RemoteAdmin commands. All other parameters 
for the region's uuid, eg regionID or region_uuid, will be removed soon. 
Please check your remote admin  scripts and make sure they are already 
using "region_id" !
