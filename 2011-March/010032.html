Ai,

There are a bunch of "challenges" with viewer 2 tiles. The login server
needs to return the URL for a web server that stores the maptiles. The web
server need to store the tiles in jpg format (i think) in files where the
name of the file encodes the location and the quality level.

The simian maptile module just uploads the highest quality tile for one
region. The simian server itself computes all the various quality/zoom
levels.

If all you want is the maptiles, you should be able to pull the map
functions out of simiangrid.

--mic
