Although I don't know all the details on how this works, I know patents are horrible thing that should not exist, and people use them in ways that should never be allowed. I dearly hope Opensim will never have to face such dangers, and that it will always stay clear of these things. I guess hoping is the best we can do, if nothing can be done to actively secure it.

Anyway Diva, that's a very nice and correct post, and hopefully people will listen do that. But imho, as long as any company has a possibility to harm Opensim, the project is unsafe. I wonder if the laws allow code owners to forbid anyone from placing patents on anything. After all, the code belongs to the devs, and no one should be able to invade it and put "rules" on Opensim without their permission. Of course other libs it uses aren't the case, if I am correct.
