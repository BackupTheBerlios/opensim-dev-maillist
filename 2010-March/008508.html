It's actually just the promised, dialog-based creation of estates,
and users as estate owners, that was lost when the master avatar was
dropped. Now, whenever a region comes up interactively, that means
on first start or when create region is used at the console, you are
asked if you want to join an existing estate or create a new one,
and for any estates that don't have an owner set, one is queried.
You have the option to enter the name of an existing avatar, or to
create a new one (standalone only). That avatar is then assigned as
estate owner.

Melanie
