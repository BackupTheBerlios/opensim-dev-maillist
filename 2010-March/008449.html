hello
i have started a new project called d4os
the goal is to build a web interface to manage a grid using drupal native
modules (like "organic groups" for groups, "services" for webservices,
"date" for events etc...).
This project contains a gui to use osservices and radmin and is using a
drupal ported version of ossearch and xmlrpc flotsam groups
The functions are separated from the input/output so this should be easy to
update to the next opensim refactor

the project page is here
http://groups.drupal.org/d4os

the first alpha release is here
http://github.com/ssm2017/d4os/downloads

this is an alpha release so there should be bugs and the job is not finished
yet, but you can begin to see how it looks like.

i need help from drupal devs to split tasks in different working groups.

if you are intersted about this, contact me by mail or join me by irc in
freenode #d4os
