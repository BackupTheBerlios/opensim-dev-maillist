If you are running a recent checkout of OpenSim you might have seen the new grid configuration options for SimianGrid. SimianGrid is a set of PHP grid services that offer an alternative to the C# ROBUST grid services that ship with OpenSim. Just like the simulator Simian, SimianGrid started as an experimental Open Metaverse Foundation project. The goal is to provide a scalable virtual world backend with a stable and well documented API that can be extended with new protocols or your own grid-specific customizations. The original code was written by Jim Radford from libOpenMetaverse / Open Metaverse Foundation. Myself and a few others have joined in on the development so far, but anyone is welcome to join. Send me an e-mail if you want to help out.

The SimianGrid project lives at: <http://openmetaverse.googlecode.com/>

ScienceSim is currently testing the SimianGrid backend on its new beta grid. If you have time, please sign up for an account on <http://grid.beta.sciencesim.com/> and try it out. That link is running SimianGridFrontend, a web frontend that talks to SimianGrid and is being developed at the same time. Any reports about instant messaging, friends, or inventory issues are especially relevant and can be posted to the scisim-discuss mailing list at <http://groups.google.com/group/scisim-discuss>.

If you are a region operator, feel free to connect to the ScienceSim beta grid to test your OpenSim 0.7pre + SimianGrid simulator(s). GridCommon.ini settings for the ScienceSim beta grid can be found at <http://grid.beta.sciencesim.com/gridinfo.php>.

When logging into the ScienceSim beta grid, make sure to use -loginuri http://grid.beta.sciencesim.com/Grid/login/
