Hi,

I just took a look at a (relatively) new region module, the
AttachmentsModule, and discovered that it implemente IRegionModule.

That interface is deprecated and NO NEW MODULE SHOULD USE IT!

Some of us work hard to eliminate modules using that interface,
please don't create new ones! The interface is intrinsically broken
and can't be salvaged.

Please implement INonSharedRegionModule and ISharedRegionModule instead.

Melanie
