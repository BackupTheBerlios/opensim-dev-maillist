Dear Tl: We have a mainlog() method that is used by UGA &S (User, Grid, Asset and Sim) to print *stuff* out to the console and this same *stuff* is written to an ascii log file. I find that examining the four logs while debugging how things work to be helpful.

Recently, I got gryc to make a .php script to 'tail' the User and Grid Server logs. I am planning on extending that idea to the Asset server as soon as it starts logging useful information.

On osgrid, are two new pages : http://www.osgrid.org/opensim/userlog.php and http://www.osgrid.org/opensim/gridlog.php

I am planning on making a similar assetlog.php as soon as the Asset server starts generating useful log messages for us all to see as we add functionality.

Charles
