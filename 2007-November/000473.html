In an effort to give unassigned bugs in Mantis more visibility, I'm
going to start sending out a weekly nag about them to the list.  Now
that mantis actually will send out email, hopefully this will make it a
bit more used as well.  The new mantis can be found at
http://opensimulator.org/mantis.

Here are the currently open and unassigned bugs.  If you are working in
one of these areas, please assign the bug to yourself to reduce the
unassigned count, as well as let others that you are looking into the
bug so we don't duplicate effort:

http://opensimulator.org/mantis/view.php?id=11

0000004 - outfits do not persist between logins
http://opensimulator.org/mantis/view.php?id=4

http://opensimulator.org/mantis/view.php?id=9
