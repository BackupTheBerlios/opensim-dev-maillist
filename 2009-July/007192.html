I have created a new git tree version for testing.  This includes
remapping some of the user ids, and a first pass attempt at preserving
all branches and tags from svn -> git.

It is web viewable here:
http://opensimulator.org/viewgit/?a=summary&p=opensim-test3

It can be checked out anonymously via:
git://opensimulator.org/git/opensim-test3

It can be checked out / committed to by developers via:
ssh://opensimulator.org/var/git/opensim-test3

Please kick the tires on this when you get a chance.  This should be
close to what the final tree looks like.  Also, if there are branches we
want to purge (or rename) we should determine that shortly.
