I would look into the collision library.  ODE can be compiled for either
OPCODE or GIMPACT.  Originally I used to compile with GIMPACT (probably
including the windows binary), but since the 0.9 release it seems like
OPCODE is getting more support and testing; GIMPACT is looking like legacy
code.  It's quite possible they react differently to the heightfield
collision calls.

-danx0r

ps I've heard rumors about an ODE/Bullet integration, using ODE for dynamics
and Bullet for collision.  Not sure what advantages that might offer.
