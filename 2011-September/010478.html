On 11-09-14 03:28 PM, Marcus Alexander Link wrote:
> how do I easily change, set or update a HUD for all existing accounts?

The easiest way (for you) is to let the wearer go get the latest version.

When the item is attached a script can check with a website or master object 
(ie. a networked vendor) if the item is the latest version. If not, the 
wearer is notified to go someplace to get the latest version.

If the only change to the HUD is the script(s) it is using, there are 
scripts that you can use to automatically update scripts in an object.
