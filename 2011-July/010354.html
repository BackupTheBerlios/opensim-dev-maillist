Looks good.  My major suggestion would be to replace the individual release columns with something like a Since column 
(e.g. since 0.7.0.2).  The vast majority of these could just be 0.6.9 (or pre 0.6.9).

Of course, actually filling in the parameters is the rub - there are a large number of them, even if one doesn't 
consider OpenSimDefaults.ini :).  The best documentation is still OpenSim.ini.example, though at least having them in a 
wiki allows people to more easily put comments by them (and makes them more findable via google).  And it doesn't need 
to be initially perfect, it can start off imperfect with some warning that it's under construction and folks can help 
fill it in over time.
