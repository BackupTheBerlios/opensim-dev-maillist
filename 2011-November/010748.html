Volker's Whisper region module binaries release won't work for current development OpenSim due to some package structure 
changes.  They will work if recompiled in the official release.

Since I did the recompilation anyway, I posted the recompiled DLLs at [1], as well as the Ice.dll and Glacier2.dll 
required.  These won't work for the OpenSim 0.7.2 release and you have to follow the normal Whisper instructions to 
configure them.

These DLLs also contain various adjustments to log messages to make debugging a bit easier (maybe).

I can't provide any support for these.
