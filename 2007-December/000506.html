Revision 2606 brings back some of the community building nature of
Secondlife.

It enables the use of the 'anyone can move' and 'anyone can copy'
checkboxes.

One important caveat though, It acts like 'Allow Friend to Edit my
Objects' permissions on that object only.

If you intend to update to this revision, make sure you back up your prim
with the save-xml command first so you have a copy of your prim (and their
permissions) before using this revision.  It makes changes to the default
'EveryoneMask' permissions that get then saved back to the database.

Also, after updating to this revision, be sure that the prim that you don't
want people to be able to edit, don't have 'anyone can modify or anyone can
copy' checked.  Objects made in previous revisions may or may not have those
boxes checked by default.
