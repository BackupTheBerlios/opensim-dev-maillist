If you're using OpenSim.32bitLaunch.exe then I expect this is some kind of bug, though I'm surprised other people 
haven't run into it before with simple cubes.

You might want to try reducing the complexity of your setup (e.g. run just one region with one physical prim) and see if 
that changes anything.  If that works, then try with more regions, etc.
