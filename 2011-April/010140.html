Hi folks.  OpenSimulator 0.7.1-rc2, the second release candidate for 0.7.1,
is now available from
http://opensimulator.org/wiki/Download

The number of changes since 0.7.1-rc1 is minimal and follows at the end of
this e-mail.  The biggest change is that
MrMonkE contributed a patch to update the MSSQL db module to 0.7.1
functionality.  However, I don't think that anybody
on core actually uses MSSQL, so any testing of this would be appreciated.

a7bc187 Added MrMonkE to the contributors. Fixed contributors.
975cebe Thank you MrMonkE for a patch that seems to bring the MSSQL data
layer up to speed with 0.7.x.
69fb1fa Stop CHANGED_INVENTORY firing twice if a notecard is edited in prim.
8f3d2f5 Fix a bug where physical objects rezzed with an initial velocity by
script do not receive this velocity.
e09f196 add the executable bit to all bundled DLLs so that these are
preserved when using git or untarring releases
under cygwin on windows
3698e43 Bump minimum required mono to 2.4.3 from 2.4.2.  OpenSim fails at
runtime below 2.4.3 due to unimplemented
ServicePointManager.get_
ServerCertificateValidationCallback()
b0bcf0c Change default ini.example port used by Freeswitch in grid settings
to 8004.
2750617 adjust freeswitch grid connector to port 8003 to agree with the
default 8003 used in OpenSim.ini.example
87eb46b Merge branch '0.7.1-dev' of
ssh://opensimulator.org/var/git/opensiminto 0.7.1-dev
7772e1e Removed stale client components: MXP and VWoHTTP.
6a3a39b recover from unhandled exception from bad rotation data while
processing entity updates in LLClientView.cs
6498701 Made things consistent between LocalInventoryServiceConnector and
RemoteXInventoryServiceConnector on
GetFolderContent.
191da19 print invalid command message to the console, not the log
f7d47bb Addresses mantis #5449 -- crashing exception in
SendInventoryFolderDetails.
9afb2e3 Removed unused libraries: RAILS.dll and dependencies.
c9f354b Removed unused libraries Castle.*
ee3d2cb Merge branch '0.7.1-dev' of
ssh://opensimulator.org/var/git/opensiminto 0.7.1-dev
bc2c55f Commented verbose debug message.
92841ea switch up flavour to rc2 in preparation
