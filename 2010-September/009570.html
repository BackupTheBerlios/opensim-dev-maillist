OpenSim can't take responsibility for monitoring changes that linux
distributions make to 3rd party packages.

In this case, the compiler was split off from the main mono
distribution, which is not possible without losing functionality.
This was done by the debian packagers, who are not obligated to
inform novell, much less us, what they're doing.

Since I don't think that you would want to fund a monthly test of
all distros to find such issues, we can only react to them and
document them after the fact. Since we have a Wiki, anyone can
document it.

Personally, I would not notice, since I use mono compiled from
source on CentOS.

Melanie
