Just put some notes in the script comments that the increased threat level
is required, and distribute it anyway. Those who want to run it will adjust
threat level on their sims accordingly.

That said, it has been my experience that despite a wealth of osFunctions
that return various interesting URL and URL components relevant to the
simulator where they are called, none of those functions actually return
the HG URL. Nor have I been able to create a derived function from the that
does so.

If you manage to do so, I'd love to know.

Then I could write script that would register the HG URL with a central HG
URL repository and produce a script that would be broadly useful for
navigating between such registered HG destinations.

Good luck though, been working/begging/praying for this functionality for
years.
