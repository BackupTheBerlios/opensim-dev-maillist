Hello Blake,

There are some ossl functions wich can retrieve grid info like the
hypergrid url. http://opensimulator.org/wiki/OSSL_Implemented

For the login url there is this function:
http://opensimulator.org/wiki/OsGetGridLoginURI

The function needs a certain threat level to function:
http://opensimulator.org/wiki/Threat_level

So it is possible but the script will only work if it has the correct
threat level permissions.
