Which begs the question of why half of the services in the Grid server 
convert the external host name to IP address and half don't...
Not just the Grid server. This happens a little all over.
If all the pieces would use whatever comes in external_host_name (domain 
name or IP address), only to convert them when absolutely necessary, 
like for comparisons, this would be a non-issue.
