Tonight I pulled together linkage between svn and mantis upon a request
this afternoon.

For committers, it's pretty simple.  If you include the string "mantis
#1234" somewhere in a commit, it will append the text of that commit
into that mantis item as an update.  If you include the string "fixes
mantis #1234" it will add that to the mantis, and mark it as RESOLVED /
FIXED.

This should make life a little simpler in closing mantis items.
