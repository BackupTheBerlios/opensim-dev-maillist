Hi,

the current Freeswitch module is deprecated. There was interest in
rewriting it for 0.7; if you are interested, please contact me, as
the other interested party has dropped the matter.

In 0.7, Freeswitch should not be handled by the region alone, but
rather should be a service, for which a skeleton already exists and
leave only call control to the regions.

Sone work has been done but it needs an interested developer or two
to finish it.

Melanie
