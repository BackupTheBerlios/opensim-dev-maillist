Hi everyone,

0.7-rc2 is now available in source and binary forms. You can get it from 
here:
http://dist.opensimulator.org/

Release notes and upgrade instructions are found here:
http://opensimulator.org/wiki/0.7_Release

We're aiming for a nicely stable 0.7 release, but some bugs are known to 
be there. Feature development and bug fixing will continue towards 
0.7.1. From rc2 to the official 0.7 release we plan to do only fixes of 
critical bugs that may show up in rc2. The official 0.7 release will be 
done within the next couple of weeks.

For those of you who test-drove 0.7-rc1 (my sincere thanks!), please 
note that I have replaced the problematic MySql driver that was included 
in rc1 with the old, non-problematic driver. What this means is that you 
need to remove the "Old Guids=true;" from the connection strings. This 
is only for those who have gone through rc1; for everyone else, ignore 
this paragraph.

Enjoy!
Diva / Crista
