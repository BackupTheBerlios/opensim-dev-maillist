I heard about "fund-raising" wanted : so we are a society working in fund raising (in italy) so we can organize a "project" for a "complete" OS bin distribution, raising funds for devs and testers (needs servers, job-time, etc...)

I heard about too many people talking about a "blanket too short" about opensim...but we know its not truth : we need only an implemented "coordination" on the project (this is a my opinion).
Finally, want ask to devs and testers to suggest a complete list of features wanted in a "work-in-progress" (for now) bin distro.
