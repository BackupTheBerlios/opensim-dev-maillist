Some new commits to the 'varregion' branch in the OpenSimulator source
repository have added better map display and teleporting for large
varregions -- you can now click on a large region in the map and teleport
to that location.

When you display varregions using the viewer's map dialog, you will now see
information for each of the 'sub-regions' of the larger region. The legacy
map system only works on 256x256 regions so each 'sub-region' of the larger
region now returns the information for the whole larger region. This means
that clicking in varregion areas outside the first 256x256 doesn't just
display 'loading' forever.

Additionally, if you click in the larger region outside the first 256x256
and click the 'teleport' button, you will be teleported to the map selected
location in the varregion.

Along with these changes, the 'varregion' branch has been updated with the
'master' branch.
