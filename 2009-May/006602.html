This relates to the code which controls the flow of data from the server to the client.  It corresponds to the 
information you'll see in the Linden Lab Second Life client stats (texture data, object data, etc.).  Sim throttles is a 
better name than OpenSim Throttles, though perhaps something like Client Data Throttles is even better.
