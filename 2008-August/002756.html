I think the main focus for the 0.5.9 - 0.6 should be testing and bug fixing. Of course anyone who is in the middle of something should be able to continue. But I don't think we really should start on anything big.

Although possible some general code cleanup. Also continuing work on fixing  the duplication in the scripting engines. 

I guess the xml3 format is also possible. But I think we want to spend some time just trying to decide what we want from it rather than rushing into coding it. 

I think the latest we should aim for 0.6 is the end of next month. When opensim will be 20 months old.
