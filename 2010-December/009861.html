> Is it possible to compile a 64-bit windows DLL of ODE that will actually work.

> been able to build a DLL that OpenSim would load.

You should be able to build one with the instructions at

http://www.onikenkon.com/ode/ODE-x64-windows_build_instructions.txt

There's also a prebuilt one in

http://www.onikenkon.com/ode/

In addition, you may also want to specify -dNODEBUG manually in the Visual Studio preproecessor properties if you 
encounter problems with assertions (whether these are serious problems or not remains to be determined).
