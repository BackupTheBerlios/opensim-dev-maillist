In master branch, the preferred syntax of the link-region command is now:

link-region <x> <y> <url> [<region name>]

Example:
link-region 900 900 http://hg.osgrid.org "Wright Plaza"

The older syntax is still supported, but it has been deprecated.

In the map search, it's still hg.osgrid.org:80:Wright Plaza
