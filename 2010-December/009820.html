spotted on the OpenSim.ini.example file (only one actually HG 
related, but both ought to be changed I think)...

a) the example FreeSwitch Service URL given uses port 8003 - whereas 
the Robust.ini example connectors uses port 8002.  Would it be 
correct to set the example to show port 8002 too to avoid this simple 
configuration mistake?

    ServicesConnectorModule = XmlRpcGroupsServicesConnector
