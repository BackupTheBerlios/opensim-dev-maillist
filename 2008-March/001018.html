mirror via tailor at http://opensimulator.org/hg/opensim-trunk.  You can
read up about mercurial here: http://www.selenic.com/mercurial/wiki/

that would like to work with hg, that's now here as well.

Git should come soon as dalien and I sort out what is going wrong with
the conversion.
