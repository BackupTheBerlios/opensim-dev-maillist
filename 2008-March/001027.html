Johan from 3Di here. I hope our merge doesn't create too many problems. 
The only thing we are aware of at the moment is you must update both the 
region server(s) and the grid server, otherwise things won't work.

Some of the code is in ThirdParty/3Di for the time being, but we are 
donating all code to OpenSim, and it is under the same license as the 
rest of OpenSim. Once the code has been tested and accepted it can be 
moved to a better home.

We are still developing this code, but we think it is stable enough to 
show to you. Hopefully you have comments and will help us improve it. 
Our tentative test results can be found here:
http://geocities.com/milbus/opensimtesting.pdf
