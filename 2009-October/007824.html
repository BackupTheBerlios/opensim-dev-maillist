Of course, BSD-licensed viewers don't suffer from this restriction, so
please feel free to look at and improve BSD-licensed viewer source code.
While the current BSD-licensed viewers don't necessarily implement all of
the same functionality as the LL viewer, they do offer alternatives for new
developments like desktop sharing.

One thing you could do with 3Di Viewer "Rei" would be to create a "mash-up"
with the 3D viewer and a 2D Java VNC applet inside the same web-page, with
the VNC applet being controlled from in-world via JavaScript. This isn't
quite the same as desktop sharing inside the 3D view, but it may be one way
to approach the problem.
