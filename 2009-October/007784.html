Hi folks,

The OpenSim 0.6.7 source release packages are now available from

http://opensimulator.org/wiki/Download

As usual, the caveats about alpha level code and minimal (or no) documentation 
apply, though I did make some notes on the ROBUST wiki page about which changed 
need to be made for a 0.6.6 to 0.6.7 migration.

Some bugs were reported against this release in Mantis but nothing that appeared 
to be absolutely showstopping (i.e. no bug prevented a correctly configured 
server from starting up at all).  So in line with previous practice the release 
went ahead.

I'm hoping the usual good people will soon produce binary packages and installers :)

This release was taken from the 28th September trunk code so it does not 
incorporate all the recent excellent texture and protocol work.  Therefore, I 
think that it may be a good idea to have a 0.6.8 once this code has proven 
reasonably stable.
