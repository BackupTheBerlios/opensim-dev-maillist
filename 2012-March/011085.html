>     Hi folks.  OpenSimulator 0.7.3.1 has just been released [1].  This is a minor point release on 0.7.3 which contains
>     a few bug fixes (e.g. osNpcLoadAppearance() attachments regression fix, off-by-one scripting error report fix, MSSQL
>     database plugin updates).
>
>     It also contains a few new minor features, such as the ability to restrict hypergrid foreigners from visiting
>     certain regions on a simulator with Hypergrid enabled.
>
>     For more details, please see the release notes at [2].
>
>     I anticipate that this will be the last 0.7.3.x release.  The next would be 0.7.4.  If this follows historical
>     precedent this will probably be in about 4 months or so.
>
>     Thanks to everyone who contributed to this release, whether with code, testing, documenting or helping people with
>     all the questions involved in using this complex piece of software :)
