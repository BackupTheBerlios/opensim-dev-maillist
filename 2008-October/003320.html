I'm looking at this as the Web.
- The hyperlinks I have now on the map are equivalent to links on web 
pages. I actually like the map, it gives a feeling of places that are 
available from any region. When I first started with SL I spent a lot of 
time looking at the map. The hypergrid allows region admins to extend 
the map beyond the one from the grid they are connected to.
- There ought to be something else that is equivalent to typing an URL 
in the address bar. That's not done yet.
- There also ought to be something else that is equivalent to 
server-side scripting on Web servers redirecting web agents -- that's 
the osTeleportAgent functions, I think.
