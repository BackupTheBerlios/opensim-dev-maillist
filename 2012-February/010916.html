On 12-02-25 04:02 AM, FoTo50 wrote:
> I'm responding to quite a bit of an old issue here asking if there has
> anything moved on in the recent versions of OpenSim. Working on a
> webinterface for OpenSim (jOpenSim, component for Joomla), I'm mainly
> looking for a way to change:
>
> ShowCase (SearchURL is used here in Imprudence/Kokua and Singularity Viewers)
> Create Event

You can send out a number of things in the login response packet. You may 
have to check the viewer source code to see what you can send. I used to 
send helperURI, currency string, and maximum group count.

It would be useful to be able to set some of those settings via a .ini file 
but that isn't supported yet.

> Partner

The partner information is just a field from the database that is shown in 
an avatars profile. There is nothing to change in OS for this. It has to be 
set externally.
