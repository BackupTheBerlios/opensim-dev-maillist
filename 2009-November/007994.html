Today I implemented an oar merge facility on OpenSim trunk.  A merge can be 
specified using a --merge switch like so

load oar --merge my.oar

more details at

http://opensimulator.org/wiki/OpenSim_Archives#Usage

The merge will leave the existing objects, terrain, region settings and land 
parcels in place.

I'm not sure how useful merging is without a way to save parts of a region 
(either via parcel or possibly owner id).  Feedback appreciated as always.
