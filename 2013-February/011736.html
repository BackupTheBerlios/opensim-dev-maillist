This to me looks like some kind of network routing issue, lowering the draw
distance probably reduces the amount of data requested during the teleport,
If i had to guess I would say this is probably an issue with the router
being used, probably the NAT table is failing in some way, could also be
older version of UDP, it is possible a firmware upgrade could help if there
are any available, it might require a new router, however I can not
guarantee this is your problem, but from my experience this sounds very
much like some kind of networking issue to me.
