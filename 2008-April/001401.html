Hi everyone,

I have setup an opensource project for the Perl UGAI at sourceforge.net
project home page url:
http://openugai.sourceforge.net/
at sourceforge:
http://sourceforge.net/projects/openugai
there is also a live demo site:
opensim.wolfdrawer.net <http://opensim.wolfdrawer.net/user.cgi>

An advanced version of Perl UGAI has been imported.
* The project focus on the UGAI-R communication protocol, not a particular
language.
* It a place you can share your favorite language implementation of UGAI.
* It also concerns about introducing Web adventages(apache, social network,
data portability, ...) into VW.

Any other language or better implementation is welcome!
Hope "OpenUGAI" can keep good relationship with "OpenSim".
