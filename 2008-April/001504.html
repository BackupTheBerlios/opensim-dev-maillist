two i know about:

-background will cause OpenSim.exe to run without console
-inifile foo.ini will cause OpenSim.exe to use foo.ini instead of 
OpenSim.ini as config file
-gridmode runs OpenSim in grid mode (as configured by the [network] 
section of the config file)

ok, that was three. ;-)
