First, I have my OSGrid hat on for a minute, as opposed to by OpenSim core hat.

We have a couple of statistics that are displayed on some grid loginscreens, such as "unique visitors last 30 days" and "online now".

I really like the idea of displaying some useful statistics for OpenSim as we scale up usage of this wonderful software and wonder if maybe we can add a few more counters that can be displayed with useful data. 

Maybe things like inventory requests last 24 hours along with failed requests. Or said in a more general way, succesful and unsuccesful actions taken by the software that let us glean some data about how we are doing, logically speaking.

This is mostly a UGAIM question as opposed to a region question and affects grids that are trying to intuit our progress.
