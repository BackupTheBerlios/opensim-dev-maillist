> I installed Tortoise on my PC but everytime i try to import the 
> Opensim trunk it ask for a login/password.

I am using Tortoise for quite a while without any problems.
Using "import" tries to send your current directory to the repository, 
and thus certainly asks for a password.
You want to use "checkout" or "update" to get the code from the 
repository to your local drive.
