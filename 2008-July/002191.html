Firstly apologies for changing the interface for IPlugin without notice.
I incorrectly assumed that since no one was using it, it was safe. I
failed to account for 3rd party modules.

In the future I'll make sure to announce changes to interfaces properly.

I have opened a bug for adding constraints to the current PluginLoader:

http://opensimulator.org/mantis/view.php?id=1647

This will solve the "Critical Plugin" problem. It should also solve the
"Exclusive Plugin" problem, since we count on Mono.Addins to know
exactly how many plugins are associated with an extension point.

Comments on improving any part of the loader are appreciated.
