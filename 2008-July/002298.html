> I don't see any reason to worry.  To me OpenSIM is about server

> want to build "products" can do so.
> 
> Marketing to me is about products.  If you identify "marketing

> to build virtual worlds some of which may be offered as products.  A

+1.  As our front page states, OpenSimulator aims to be a generic 3d 
virtual worlds platform.  The Second Life aspect is almost like a 
reference implementation of a particular world that serves as a useful 
trellis for growing the platform.  It's quite possible that as 
OpenSimulator matures, the Second Life specific parts will bud out into 
a separate project.
