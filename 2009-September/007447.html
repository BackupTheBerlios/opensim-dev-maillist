The 'interface v5' part refers to the interface protocol between the region simulator and the grid (and defacto, between 
region simulators).  It doesn't have anything at all to do with the OpenSim version (0.6.6).

In principle, if you are running from a source repository update then the git version should be tacked on the end of 
OpenSim 0.6.6 (e.g. 0.6.6 239a1).  This should also be true if you are updating from the SVN mirror (e.g. 0.6.6.10108). 
  If this isn't the case then this is arguably a bug.
