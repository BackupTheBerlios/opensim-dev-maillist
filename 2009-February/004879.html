Hello,

Some of you may find build failures when upgrading from r8218 or
older. Please make sure to rebuild *all* the binaries by first running
`nant clean`, removing all artifacts from the bin/ directory or simply
performing a clean checkout.

In r8219-8224 the IInventoryDataPlugin, IUserDataPlugin and
IAssetProviderPlugin interfaces were moved from the OpenSim.Framework
namespace to the OpenSim.Data namespace. IAssetProviderPlugin was also
renamed to IAssetDataPlugin. Any binaries with references to these
DLLs need to be rebuilt.

There is already one issue filed with regards to this:
http://opensimulator.org/mantis/view.php?id=3074

Thank you,
Mike
