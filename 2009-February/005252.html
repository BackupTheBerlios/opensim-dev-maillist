OpenSim can _not_ provide a core module that links with Tahoe.

Tahoe is GPL, not LGPL, so anything using it must be GPL as well. If 
you create a forge module that is GPL, link to Tahoe from there, and 
a user installs that module, they will be fine. OpenSim can't touch 
Tahoe with a 10 ft. pole.

Melanie
