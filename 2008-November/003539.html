Hello.  I created a .deb of 0.6.0, should anyone be interested.  I've
tested it on fresh Ubuntu Hardy and Intrepid installs; haven't tried
Debian yet, but will tomorrow.  To give it a go, add:

    deb http://pkg.knifejaw.com/ubuntu opensim releases

to your /etc/apt/sources.list , then:

    # apt-get update
    # apt-get install opensimulator

It's not quite to Debian Policy Manual standards (e.g. everything's in
one big dir), but I'll improve it over time.  Everything will be
installed in /usr/lib/opensim .  The only modification I made was to
change the various *.exe.config files to add "/var/log/opensim/" as a
prefix to the log filenames to keep them out of the bin dir.

I was able to run a couple of regions on OSGrid with this package,
but if you do want to try it be sure to back up everything on your own
beforehand just in case.

Let me know if you find it useful or if something's broken.

-Coyle
