I think maybe x64 compatibility checking might be a good goal for this current 0.6 version number. There's nothing in OpenSim which prevents us from being properly x64 supported - only some of our dependencies. I have personally a desire to get as much 'managed' code as possible to prevent a large number of these issues. If we can get the default storage and physics engines fully managed, that might go a long way to helping - although the only decent managed physics engine is BulletX.
