For those on Linux (or using nant on windows), I've integrated another
nant target that will trigger the unit tests in the tree.  Type "nant
test" to kick those off.  You'll see the output on the console.

While we don't have a lot in there yet, making it easy for developers
and users to run these tests locally will hopefully help increase their
use and have new ones coming in.
