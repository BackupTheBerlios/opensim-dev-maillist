You do it with Region modules, which are user-level plugins. Here is the 
pointer. but I have a feeling the code itself is outdated and won't work 
with the latest versions of OpenSim (does anyone have an updated version?)

http://opensimulator.org/wiki/Region_Modules

The example doesn't show, but these modules can do just about 
everything, including opening network endpoints (say, HTTP handlers) 
that can receive external requests, and just about everything else you 
can think of doing. There are a few dozen region modules in OpenSim core 
itself, so you can find many examples there.
