I find this to be confusing too - in the past when I've tested things out on the Linden grid, for instance, some 
operations related to groups worked fine if that group wasn't the active group.  However, this may not be consistent 
across all Linden operations or perhaps the situation has even changed over time.

In either case, I don't think there's any problem with having an option for different behaviour in OpenSimulator, as 
long as the default matches the expectations set by the Linden grid.
