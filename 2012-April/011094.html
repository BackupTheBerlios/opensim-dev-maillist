This could be any number of issues,
  It is important to know how the walk animation is triggered...

  If it is triggered by an AO, and your simulator is busy, or you crossed a
region border,
     it could be a script issue..

  If everything works with a different set of animations, it could be the
new walk animation priority was set differently when uploading.

  If your network is especially busy, it could also be a viewer issue.

Almost all the viewers allow you to display the list of animations being
played in
floating text above the avatar, (Advanced>Character>Animation Info).

That's a good place to make sure what you expect is actually what is going
on...
