Hi folks.  The first release candidate for the next version of OpenSimulator is now available as 0.7.4-rc1 [1].  Draft 
release notes are at [2].

As ever, only functional regressions from previously working functionality will be considered for fixing, where one 
possible outcome is WONTFIX.  Please file any of these bugs under the 0.7.4-rc1 release option in Mantis.  Please report 
all other bugs against master as normal.

Many, many thanks to all the developers, testers, documentation writers, administrators, builders, scripters and 
community members who are contributing towards these releases.  None of this would be possible without your work.

[1] http://opensimulator.org/wiki/Download
