Aloha,

I would recommend that you take a look here...
http://lib.openmetaverse.org/wiki/Documentation#API_Documentation

In section 5..you will find a reference to TestClient,  which is a fully
functional bot control application.
I have been using variants of this code for a number of load test
applications.

There isn't a lot of documentation, but the code is pretty straightforward.

I'm not sure what sort of load testing you intend, but this client is known
to degrade
performance during login...  If you want to test BURST capabilities, you can
log in significant
numbers of BOTS at once...  Otherwise...  logging a few bots in at a time
and letting them accumulate
seems to be the best true measure of Sim capacity.

D
