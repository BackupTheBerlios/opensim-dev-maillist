That day kind came almost a year ago, when Ryan and some other folks
wrote https://github.com/realXtend/taiga-wizard :)

It is a gui app that at least writes opensim.ini, I'm actually not sure
if it can read existing confs for editing. Nothing too fancy, was
whipped up quite quickly by the guys at Adminotech, but has been very
useful (and they did return to it later to improve etc., has been in
actual use a lot).

There's a screenshot:
http://wiki.realxtend.org/index.php/Getting_Started_with_Taiga#Configuring_with_Taiga_Config_Wizard

It being for Taiga means that it has things both vanilla Opensim, and
Cable Beach and ModRex modules. Is probably simple to remote the add-on
specific parts if someone wants it for plain Opensim. Is made with Qt
and C++. At least the GUI etc. perhaps gives some ideas.

About PHP tools, SimianGrid Installer GUI seemed to be a nice php+html
wizard for creating the confs there.
