> Umar,
>   Are you using Simian or Robust to create your accounts? If the grid
> required authentication (email, captcha, etc) you will probably need to work
> with the grid operator to get test accounts. On Simian, it's easy to create
> test accounts if you can disable the authentications temporarily.
>
>  As for logging them in, I suggest not putting more than about 25 bots in a
> single TestClient instance. There are many functions in TestClient which
> iterate over all bots sequentially performing an action and you will find
> them not responding to scene updates in time if you have too many in one
> executable.
>
>  We are planning to make our modified TestClient available but it is not at
> this time. It's actually older than the released TestClient from libomv but
> it has a few commands for making bots perform interesting actions
> independently rather than all bots doing the same thing.
>
