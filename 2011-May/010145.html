On 30/04/11 18:55, john felipe urrego mejia wrote:
> Hi, please let me know what is the best way for update for standalon with opensim 071 rc1

If you're using mysql, then it's simply a matter of installing rc2 and copying over your config files from rc1 (in this 
case, nothing has changed between the releases).

If you're using sqlite and care about your data, then you should copy over all the bin/*.db files from rc1 to rc2 as 
well as the config files.
