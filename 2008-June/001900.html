On Thu, Jun 19, 2008 at 08:33:50PM +0200, Lc wrote:
> Hi,
> 
> what is the difference between
> 
>    -   http://opensimulator.org/svn/opensim/trunk/OpenSim/
> 
> and
> 
>    -
> 
>    svn co svn+ssh://USERNAME@opensimulator.org/var/svn/opensim/trun
> 
> 
> 
> Where should I get a dev. login/pass ?

They are the same subversion tree.  The first one is accessible by
anyone, then second is the ssh url used by core developers to push
changes.

The process to become a core developer involves contributing significant
contributions to the project via mantis patches, and building up trust
with the existing core developers (as anyone with core access can change
any part of the project).

If you are interested in contributing to OpenSim, please open
bug/feature mantis items and post patches to those to fix the issues you
are interested in.
