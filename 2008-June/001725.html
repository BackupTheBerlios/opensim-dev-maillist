These are all good questions.  To some extent the best thing is just to 
try some of them out - I anticipate that the M server isn't required in

afraid you'll have to ask Teravus directly (he is on IRC a lot).

since none of the servers message each other when they're started up. 
I'd be surprised if the same isn't true for the messaging server.
