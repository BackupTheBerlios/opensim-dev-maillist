I started toying with pCampBot last week when trying to track down the
deadlock issues, and started in yesterday on molding pCampBot into a
more formal set of regression tests.  I expect I'll need a few more days
of experimentation before making any substantial changes to it, but
wanted to let people know that pCampBot is going to change in form to
give us our first regression testing suite.  The end goal is to make it
reasonably easy to verify if a new changeset is any good.

Once I've got the ground work in tree, I'd like to get discussion on the
right test cases we want to force out.
