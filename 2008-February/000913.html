I'm pretty sure you'll find (though I'd be interested if others disagree) that the BSD and GPL licenses really don't play nice togethor.

How content is licensed is a separate issue actually.  One of the interesting things about SL was that they let you own the content you create. But that's separate from the license for the software used to access it.  For instance while the viewer is GPL'd it can be used to access a proprietary system like SL or a BDS licensed system like OpenSIM.
