Hey

This is just a notice that several of the periodic processes on the
opensimulator.org box broke.   I've fixed a few of them.

Three noteworthys:


   1. http://docs.opensimulator.org/ is now updated again every 10 minutes
      - This keeps the doxygen docs up to date
      2. OpenSimulator.org's wiki search box page indexer is running again
   (permission errors prevented a symbolic link of the indexes when they were
   building)
      - This issue prevented the index from updating with new pages and
      edits
      3. OpenSimulator.org's search query processor is running again (looks
   like someone updated to a newer version and didn't update /etc/rc.local with
   the new directory)
      - This means that full text search should be working again on the
      OpenSimulator.org wiki.


Try the search box (use the 'Search') button on OpenSimulator.org and let me
know if this improves search results (It should!)

Sincerely

Teravus
