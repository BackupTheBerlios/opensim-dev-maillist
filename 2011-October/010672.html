IMO, display names are nothing but cosmetic. It is fun to have, but
does not actually add anything to the functionality of OpenSim like
the implementation of collada mesh, or NPCs did.
This automatically gives display names a lower priority than for
instance implementing remaining LSL functions, fixing LSL functions
that are not fully implemented or not working correctly, fixing
stability issues and a lot of other stuff that i failed to mention.
So sit back, be patient. It will eventually be added, but it will take
some time before that happens.

P.S. Having a display name does not only affect the name over your
head, it also affects groups, profiles, and a lot more which will need
to be changed for the display names to work properly. As you can see,
this is not something that can be added in just a few minutes...
