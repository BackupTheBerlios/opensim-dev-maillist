As you know from my previous inputs I am keen on using te URL format 
that Diva has been refining, but with the "cachable" data being in 
the key=value format for the properties like display name and other 
things which might be required or optional.  Positional formats as 
more properties are added, or customised variants are introduced, 
could become difficult.

>...though more to handle the case where the lookup URL had gone away 
>or no network was available.

Yes, that has not been mentioned before. We have some use cases where 
OpenSim is run disconnected from the Internet completely.  OpenSim 
needs to always work without depending on an hosts to a closed or 
isolated network.  Firewalled situations and strong corporate or 
institutional controls on external access are also likely for many 
applications anyway.
