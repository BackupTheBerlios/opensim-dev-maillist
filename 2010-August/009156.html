Hi folks. FYI, master now has an extra --merge option for "load iar".  With this option, when an IAR is loaded it will 
be merged with the existing folder structure where possible, rather than always creating new folders.  This can be handy 
for restoring user inventory backups.

See http://opensimulator.org/wiki/Inventory_Archives#Upcoming_functionality for more details.
