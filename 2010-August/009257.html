in master.. which makes sense as one of the last of the old style 
config files used.  But the file did allow initialization values for

where the provider wanted to ensure that any newly created region had

whole sim's regions be a rocky and snow zone), or other things 
different to defaults.

Have all the values that previously could be specified by
