Hi,

the default values are now the hardcoded defaults in the
EstateSettings class.

Any automated delivery system worth it's salt should prepopulate
this in the database anyway, so the implicit creation is only
relevant for manual region bringup, typically home or very small
grid solutions, where the defaults can very well be set inworld by
the owner after creation.

Melanie
