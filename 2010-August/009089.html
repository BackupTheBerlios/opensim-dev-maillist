bin/addon-modules/*/config/*.ini

Wifi.ini in there it might all work fine.  The current binary Diva 
distribution places the modules directly in bin just now - but it 
looks like Diva is planning in her master (not yet released).

The Wiki page and a generic setup that can be placed into any type of

this.  Diva can advise when would be a good time to help test and 
help her with this module which I think will be useful for some small 
to medium grids.

even be sensible one day?
