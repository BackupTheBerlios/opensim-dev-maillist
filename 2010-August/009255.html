Still -1.
If operators choose to credit the project, the words/logo "Powered by 
OpenSim" should be placed discretely somewhere in the welcome page (if 
it exists), and not thrown at end users as the first thing they see of 
these worlds.

The branding of the OpenSim platform is a good thing... if done right. 
When done wrong, it is a bad thing, and we end up with headlines like this:
http://www.hypergridbusiness.com/2010/08/how-to-get-ripped-off-in-opensim/
These headlines happen because people so far have a hard time 
distinguishing the platform from its uses. We need to be *very* careful 
as we move forward, can't be sloppy here.
