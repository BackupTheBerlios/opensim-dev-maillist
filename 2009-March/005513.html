Dear Rich:

On OSGrid is a region called "Metaversity Campus", which was recently created by BlueWall Slade for the purpose of virtual education. Another region currently being used for education on OSGrid is Dradis. Additionally, there are a number of UC Irvine regions. 

You might find it interesting to set aside some time to both examine these regions and consider using them in a seminar at the conference being planned. I would suggest the mere use of these regions would constitute a significant "preso".

There is currently a C# class meeting twice a week with 12 students on OSGrid and it is going very well. The students are even asking for additional homework, which I would submit is a good measure of the success of a course.

Charles Krinke
