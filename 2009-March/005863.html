The easiest thing to do here is to delete the NHibernate database
plugin if you're not using it.    It's located in /bin/ and there's
the world NHibernate in all of the components.

The best thing to do is to install the Oracle client
apt-get install libmono-oracle2.0-cil   (*replace with your mono version here*)
